Ошибка в методе isTeenager(). Он возвращает true только для всех возрастов меньше 19 лет.
Он не учитывает включительно само число 19 и учитывает детей младше 13 лет.
Под тинейджер подходит только от 13 до 19 лет включительно.
Правильная версия метода будет -

public class Person {
    public static boolean isTeenager(int age) {
        return age >= 13 && age <= 19;
    }
}

Ошибки тестов -
1) 19 лет ожидание "true", вывод - "false" т.к. isTeenager() не учитывает 19 лет включительно.
2) 10 лет ожидание 'false", вывод - "true" т.к. isTeenager() возвращает true для всех возрастов меньше 19.
